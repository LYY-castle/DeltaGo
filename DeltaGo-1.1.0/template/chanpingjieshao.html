DeltaGo

一款融入人工智能系统的

DeltaGo是基于AI人工智能技术为核心所开发的全球理财系统，简称：智能财富机器人。由现代华尔街风云人物Dr. Daniel Gurley （丹尼尔·格利博士）领衔全球著名交易师、资管团队以及诸多交易指标发明人亲自共同打造，成功建立融入AI（人工智能）计算方法的开放化、自动化、智能化的资产策略系统以及移动互联网应用APP。全球投资人均可通过DeltaGo在充满风险的金融市场中获得最安全稳定的投资回报。

Monte Carlo Tree Sea

采样搜索系统

用采样结果的频率来计算真实概率问题的算法，用于解决大规模市场规划问题。

Rollout Policy Network

智能提取网络系统

自动化智能一键提取全球金融市场特征 + 线性 softmax 分类器，用以解决MCTS随机采样所带来的信息格局不足，并使用SL（深度卷积网络）使系统实现投资监督与学习的功能，并更加智能与快捷。

RL Policy Network

强化学习网络系统

通过 SL 过程训练好的 AI 自我对冲。不关注对冲过程，只看AI虚似主角对冲结果。如果盈利，给予+1的奖励，反之给予-1 的惩罚。利用返回的奖惩值指导网络权值的调整大小与方向。此过程已可以实现当前如何进行交易的方向及产品的选择。

Value Network

价值网络系统

价值网络与RLPN不同，不为系统输出每次交易的方向，而是所有产品当前进行交易的价值计算。在自我对冲过程中，随机抽取市场信息并评估交易价值与状态，用来重复训练价值网络。

Markov Decision Proc

决策系统

使用高频交易进行采样且不成交，重复枚举之后的市场策略并不断计算平均回报，获得状态价值并使成功概率无限接近价值网络。

Intelligent Strategy

智能策略演算系统

每分钟进行上亿次不间断演算，结合MCTS、价值网络以及MDP的演算结果，并采集全球信息数据进行EM算法（最大化期望算法）、遗传算法（不断复制迭代并淘汰适应度低的函数值）以及粒子群算法（通过跟踪两个极值来更新自己并通过迭代找到最优解），最终达成完整的智能投资策略组合。

Defense System

防御系统

最新德尔塔智能防御系统：海龟交易20年测试中已证实超越奥米伽系统及阿尔法系统的全新技术防御系统。高频风险抵御机制全面保障投资安全。
